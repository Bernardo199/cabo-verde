# 🇨🇻 Cabo Verde Hoje

Website informativo sobre Cabo Verde com foco em tendências, cultura e pesquisas populares.

## 🌍 Sobre o projeto

Este projeto é um site simples criado para mostrar informações sobre Cabo Verde, incluindo:

- Cultura e ilhas principais
- Tendências do que as pessoas estão a pesquisar
- Conteúdo educativo e informativo

O objetivo é praticar desenvolvimento web e criar um projeto de portfólio.

---

## 🛠️ Tecnologias usadas

- HTML5
- CSS3
- JavaScript

---

## 🔥 Funcionalidades

- Página informativa sobre Cabo Verde
- Lista de tendências populares
- Filtro de pesquisa em tempo real
- Design simples e responsivo
